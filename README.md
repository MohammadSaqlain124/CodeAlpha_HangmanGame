# 🎮 Hangman Game (Python)

This is a simple text-based Hangman game built using Python as part of the CodeAlpha Internship.

## 🚀 Features
- Random word selection
- Limited incorrect attempts (6)
- User input validation
- Win/Loss conditions

## 🛠️ Technologies Used
- Python (loops, conditionals, lists, strings)

## ▶️ How to Run
1. Download the file
2. Run: python hangman.py
